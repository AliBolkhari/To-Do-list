# To-Do-list
A simple and responsive To-Do List web application built using HTML, CSS, and JavaScript.
This project allows users to manage their daily tasks by adding, completing, and deleting items from the list.

# Features
Add new tasks
Mark tasks as completed
Delete tasks
Clean and simple user interface
Responsive design
